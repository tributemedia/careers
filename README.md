# Careers - Drupal 7 Feature

## Summary  
Add a careers page to a site. The careers page includes a listing of all open postions, with a link to each node. There is a default banner image, that can be changed upon request, and a short bit of content that should be updated to site specifics. Each node will link to a "Submit Resume" form and autopopulate the position field.

The feature includes a career content type, view, some sample content, node template, and webform.

## Install
Upload the module to your site (usually in the /sites/all/modules/features/ folder), and enable just like any standard module.
